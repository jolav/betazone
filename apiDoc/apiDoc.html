Betazone Version
Description: Returns the actual version
 HTTP Request: GET https://m.d100.top/a/version
 Example:
  ->   https://m.d100.top/a/version 
        {
          "version":"0.5.1"
        }

Ping
Description: Makes a ping to the server
 HTTP Request: GET https://m.d100.top/a/ping?min=0&max100
min must be between 0-1000
max between 1-1000
maxmust be greater than min
 Example:
  ->   https://m.d100.top/a/ping?min=0&max=1000 
        {
          "ping": 57
        }

Random
Description: Returns a random surname out of almost 90000 possibilities
 HTTP Request: GET https://m.d100.top/a/random/name
 Example:
  ->   https://m.d100.top/a/random/name 
        {
          "name":"MULHOLLAND"
        }
